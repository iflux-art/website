# VitePress风格组件展示

本文将展示Project中所有VitePress风格的组件和效果，并详细说明它们的调用方式和展示效果。

## 渐变标题效果

渐变标题是VitePress风格的一个重要特性，它UsageCSS渐变效果使标题文字呈现出渐变色彩。

### Usage方法

```tsx
import { GradientHeading } from '@/components/ui/vitepress-styles';

<GradientHeading className="text-4xl font-bold">
  渐变标题示例
</GradientHeading>

// 可以指定不同的标题级别
<GradientHeading as="h2" className="text-2xl font-semibold">
  H2 渐变标题
</GradientHeading>

<GradientHeading as="h3" className="text-xl font-semibold">
  H3 渐变标题
```

## 无下划线链接

VitePress风格的链接没有下划线，这使得页面看起来更加整洁。

### Usage方法

```tsx
import { NoUnderlineLink } from '@/components/ui/vitepress-styles';

// 默认链接（有下划线）
<a href="#" className="text-primary underline">默认链接（有下划线）</a>

// VitePress风格链接（无下划线）
<NoUnderlineLink href="#" className="text-primary">
  VitePress风格链接（无下划线）
</NoUnderlineLink>
```

### 效果展示

  <a href="#" className="text-primary no-underline">
    VitePress风格链接（无下划线）
  </a>
</div>
