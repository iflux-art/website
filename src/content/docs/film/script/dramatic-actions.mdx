## 什么是戏剧动作

动作是戏剧要素之一，指为了某个目标和任务而采取的行动，是戏剧中情节的发展和势态。

没有目的性，不能引起观众的注意，或是观众不觉得有什么目的，这都不是动作。

影视剧看来看去，就是看两点，一是出事了，二是要出事了。明示或暗示给观众，这个动作是有目的性的，为了某个目标、任务、意图的行为，才是动作。

动作线指动作或故事情节发展的轨迹或路线。动作线的确立，使剧更有悬念、逻辑、和方向。

## 戏剧动作的来源

没有动作，人物就是一盘散沙，观众就迷失。有了动作，再给这个动作设置障碍，这就是戏。

1. 动作来源于前戏。如有的剧，一上来，人物就喘着粗气在跑，原因是前戏让他跑。
2. 动作来源于外向型性格。人物性格是有差异的，有外向的，有侵略性的，让这样的人出第一拳，就能引起一系列动作。
3. 动作来源于人物或人物关系的能量。我们观察生活，有很多人的动作，来自于自身的能量，它要释放出来，水从高处流下来，比如小孩子吃饱了，浑身是劲，就要运动，累了困了，就吃就睡了。要找到人物的能量。
4. 动作来自于人原始的本能和欲望。如看到美女，看到金钱等，要找到调动欲望的诱饵。
5. 来源于对外力打击的反弹。作用力和反作用力是相等的。
6. 来源于利益的诱惑。惰性强的人前面有个金砖，就跑过去取。
7. 来源于危险的恐惧。后面有只老虎，都跑，都躲。
8. 来源于自身拥有的资源。有资源的人，可能招来外来的力量，比如有钱人，招来人杀他；漂亮，招来人爱；有权，招来人利用。
9. 来源于人对美好事物的向往。想美容，想做好头发，想做善事等。
10. 来源于对基本权力的抗争。基本权力被打压，被剥夺，被侵犯，都会抗争，都产生动作。
11. 来源于人与人的博弈以及博弈的强弱消长。

## 单线故事动作的设置

一个故事有很多人，但这些人都是围绕一个人的，只讲一个人的故事，别人的动作只能与他一个人有关系，与别人有关的，不去阐述。

1. 障碍设置。要有层次，不要没有层次，混混沌沌；有跌宕，要在心理上有震荡；有曲折，不要一眼看透；有节奏，有快有慢，不要机械重复；有递进，不要等值发展，强度要不断地加强。
2. 进程设置。不要总是失败，也要有成功，要有喜有悲。
3. 要分段落，在总任务总动作中，可以有段落，划分出章节。

## 复线故事动作的设置

小说电影如果是对打，电视剧就是群殴，群殴的前提是一号人物动作的同时，其他人物也在动作。

1. 主线为主，兼做副线，副线为主线服务。
2. 主线与副线平行，价值独立。
3. 主线与副线树型发展，分叉发展。
4. 河流状发展，很多副线最终归一。
5. 藤缠树型发展，有主有从，又是并行的，还可以转接，跳来跳去。

## 副线动作的连接点或转换

1. 自然时间的连接转换。不同的时间，镜头就对不同的人。
2. 故事逻辑的连接转换。有发展过程。
3. 形势相似的连接转换。不同线索发生在同一地方；或有个道具，有相似的地方来转换。
4. 围绕主要人物的连接转换。主人公的成长、经历等，什么时候需要转，就转过来。
5. 平行并列。一是自然时间，二是错过时间，重复表现。

## 避免副线动作庞杂和混乱的原则

1. 突出主线。
2. 突出表现主题。
3. 服务于主要戏剧任务。
4. 围绕主要矛盾。
5. 围绕主要人物。
