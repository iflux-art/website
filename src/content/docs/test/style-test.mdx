# 样式测试

这个页面用于测试代码块和行内代码的样式。

## JavaScript 代码块

```javascript
// 这是一个 JavaScript 函数
function hello() {
  console.log("Hello, World!");
  return "Hello, World!";
}

// 调用函数
hello();
```

## TypeScript 代码块

```typescript
// 这是一个 TypeScript 接口
interface User {
  id: number;
  name: string;
  email: string;
}

// 这是一个 TypeScript 类
class UserService {
  private users: User[] = [];

  constructor() {
    console.log("UserService initialized");
  }

  getUsers(): User[] {
    return this.users;
  }

  addUser(user: User): void {
    this.users.push(user);
  }
}
```
