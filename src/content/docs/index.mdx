# 欢迎阅读斐流艺创技术文档

欢迎来到斐流艺创（iFluxArt）技术文档中心。本文档旨在为开发者、贡献者和用户提供全面的技术指南，帮助您更好地理解、使用和贡献于我们的项目。

## 文档结构

本文档采用 [Diátaxis 文档框架](https://diataxis.fr/) 组织，将内容分为四个主要类别：

1. **教程 (Tutorials)** - 学习导向的材料，帮助新手从零开始
2. **操作指南 (How-to Guides)** - 目标导向的实践指导
3. **技术参考 (Reference)** - 提供技术细节和规范信息
4. **解释说明 (Explanation)** - 背景知识和概念解释

## 开始阅读

根据您的需求选择合适的文档：

- **新手入门**: 请从[快速开始](/docs/getting-started)指南开始
