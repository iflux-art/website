Label 组件是一个可访问的标签组件，用于表单控件。

## 基本用法

以下是 Label 组件的基本用法示例：

```tsx
import { Label } from "@/components/ui/label";

export function LabelDemo() {
  return (
    <Label htmlFor="email">电子邮箱</Label>
  );
}
```

## 与表单控件一起使用

Label 组件通常与表单控件一起使用：

### 与输入框一起使用

```tsx
import { Label } from "@/components/ui/label";
import { Input } from "@/components/ui/input";

export function LabelWithInput() {
  return (
    <div className="grid w-full max-w-sm items-center gap-1.5">
      <Label htmlFor="email">电子邮箱</Label>
      <Input type="email" id="email" placeholder="输入您的电子邮箱" />
    </div>
  );
}
```

### 与文本域一起使用

```tsx
import { Label } from "@/components/ui/label";
import { Textarea } from "@/components/ui/textarea";

export function LabelWithTextarea() {
  return (
    <div className="grid w-full gap-1.5">
      <Label htmlFor="message">消息</Label>
      <Textarea id="message" placeholder="输入您的消息..." />
    </div>
  );
}
```

### 与复选框一起使用

```tsx
import { Label } from "@/components/ui/label";

export function LabelWithCheckbox() {
  return (
    <div className="flex items-center space-x-2">
      <input type="checkbox" id="terms" />
      <Label htmlFor="terms">接受条款和条件</Label>
    </div>
  );
}
```

## 必填标签

可以添加必填标记：

```tsx
import { Label } from "@/components/ui/label";
import { Input } from "@/components/ui/input";

export function RequiredLabel() {
  return (
    <div className="grid w-full max-w-sm items-center gap-1.5">
      <Label htmlFor="name" className="after:content-['*'] after:ml-0.5 after:text-red-500">
        姓名
      </Label>
      <Input id="name" placeholder="输入您的姓名" required />
    </div>
  );
}
```

## 禁用状态

当与禁用的表单控件一起使用时，Label 组件会自动应用禁用样式：

```tsx
import { Label } from "@/components/ui/label";
import { Input } from "@/components/ui/input";

export function DisabledLabel() {
  return (
    <div className="grid w-full max-w-sm items-center gap-1.5">
      <Label htmlFor="disabled">禁用标签</Label>
      <Input id="disabled" disabled />
    </div>
  );
}
```

## API 参考

Label 组件基于 Radix UI 的 Label 组件，并接受所有标准的 HTML label 属性。

| 属性 | 类型 | 默认值 | 描述 |
| --- | --- | --- | --- |
| `htmlFor` | `string` | - | 关联的表单控件的 ID |
| `className` | `string` | - | 自定义类名 |
| `children` | `ReactNode` | - | 标签内容 |
