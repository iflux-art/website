# 人

> 智能推荐兴趣用户，积累粉丝用户，沉淀「人」群资产

- 潜客触达：自然流量、商业流量、品牌老客。
- 看播停留：优化直播场景，丰富直播互动玩法。
- 引导转化：优化货品及利益点，增加直播间专属权益。
- 用户沉淀：引导用户加入粉丝团，老客营销。

# 货

> 万千创意好货（品质保障、优质优价、新奇好玩），爆品差异化，打造渠道专属「货」品池

- 选品：分析电商渠道爆品结构，选出适合渠道的产品。
- 测款：通过达人专场测款，将货品复用到品牌自播账号。
- 优化：分析爆款数据，不断更新迭代优化，提供给用户最适合的商品。
- 爆款池：持续增加渠道专供款，打造抖音专属爆品池，降低和其他渠道的爆品重合度。

# 场

> 短视频+直播，建立品牌经营「场」地

- 开播即是门店营业：品牌持续自播维持账号热度，提高粉丝黏性，达到日均百万的自播产出。
- 货品内容化短视频：货品内容化呈现，绑定品牌定位，打造品牌独有优质内容阵地。
- 达人短视频种草：垂直穿搭+泛娱乐达人短视频投放策略，为产品带来强种草，打造爆款产品，提升品牌影响力。
