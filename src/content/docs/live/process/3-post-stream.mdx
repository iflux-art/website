## 直播复盘

### 为什么要复盘

1. 为了帮助我们的下一场直播越来越有号召力、感染力以及销售力，把经验转化为能力。

2. 复盘总结，提升数据。

> 录播自己的直播间是一种非常好的复盘形式

### 需要复盘什么

#### 直播基础数据/互动数据

| 数据     | 说明                                                                                                                                                                                                                                 |
| -------- | ------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------ |
| 观众总数 | 决定了直播间所在流量池等级，是决定直播间流量的关键指标，有人的地方就有市场。直播间也需要有承接流量的能力。                                                                                                                           |
| 新增粉丝 | 直播间有多少人关注了你的账号，在直播推荐打开的情况下，一般新增粉丝比例达到 5% 算比较不错的数据，代表对新进直播间用户的吸引程度。单场直播转化新粉比例低于 3% 时，说明陌生用户没有被你的直播内容吸引，影响因素包括人、货、场多个方面。 |
| 评论人数 | 直播间的用户互动情况，是影响直播间人气的关键因素。一般互动比例达到 10% 算是不错的数据，低于 5% 要考虑是不是主播不会活跃气氛、不会说话，长得是不是不符合粉丝的审美，产品是不是没人喜欢，直播间是不是脏乱差。                          |
| 付费人数 | 送出粉丝团灯牌以及其他抖币礼物都算是付费用户。带货直播间付费用户数据一般参考性不大。                                                                                                                                                 |
| 收获音浪 | 直播间收获的音浪总数。带货直播间不同于娱乐直播间，一般音浪收入普遍不高。                                                                                                                                                             |

| 指标                   | 定义                                             | 说明                                                                                                            |
| ---------------------- | ------------------------------------------------ | --------------------------------------------------------------------------------------------------------------- |
| 人气峰值和平均在线人数 | 决定了直播间的人气，是能否带动货的前提           | 一般平均在线能稳定在 50 人左右，直播就有基本的带货能力，能够赚到钱                                              |
| 观众平均停留时长       | 是内容吸引力指标里最重要的一项                   | 停留时长数据越好，说明主播的留人技巧和选品都不错，通常停留时长超过 2 分钟算不错的数据                           |
| 带货转化率             | 直播间的用户下单比例，等于下单人数除以观看总人数 | 可以衡量直播间的真实购买力，也反映主播的带货能力、选主播的标准                                                  |
| UV 价值                | 单个用户给直播间贡献的价值                       | 这个数据越高，说明用户在这个直播间的付费意愿更强；有些主播的一场 UV 价值达 30，说明粉丝消费力极强，销售额肯定高 |

1. 商品点击率+转化率：进行货品调整

> 高点高转：多次返场讲解 or 产品升级迭代  
> 高点低转：调整价格 or 优化详情页 or 加强话术  
> 低点高转：话术重写 or 多次返场讲解  
> 低点低转：互动询问反馈 or 缩短讲解 or 下架

2. 人均观看时长&小黄车点击率：验证脚本话术促单、讲解引导力的强弱

3. 直播间点击率：关联直播间画面的吸引力

4. 评论高频词：可按需优化产品和福利机制

#### 直播流量来源

1. 直播推荐：直播广场免费推荐流量

2. 其他：巨量千川加热直播间，小时榜、PK 连麦、粉丝分享等外部引流多种方式进入直播间

3. 关注：通过粉丝推荐和关注页进入直播间

4. 同城：通过同城推荐进入直播间

5. 视频推荐：通过自己视频推荐，他人视频引流推荐或巨量千川选择视频加热直播间

#### 直播销量分布、货盘分析、用户画像

1. 直播间的选品逻辑，引流款、利润款、主推款分配是否合理

2. 过款流程的安排是否合理，单品的核心卖点提炼非常重要

#### 直播团队配合

| 角色 | 复盘内容                                                                 |
| ---- | ------------------------------------------------------------------------ |
| 主播 | 复盘直播中直播脚本、话术问题，产品卖点掌握情况、控场情况                 |
| 场控 | 直播中的实时目标关注，直播间热度变化，突发事件预警能力，直播复盘的组织者 |
| 助理 | 复盘商品上下架，关注直播间设备，确认发货快递和发货时间，与主播配合等     |
| 运营 | 预热视频、引流视频的准备和发布，巨量千川的投放操盘问题                   |
| 选品 | 对于品牌和商品的筛选是否合理，利润款、引流款、福利款产品的设置是否合适   |
| 客服 | 活动福利说明以及可能存在的售后问题，直播过程中回答粉丝的提问等           |

#### 直播场景复盘

场地布置、直播间背景、直播间灯光、直播设备、商品陈列等。
