# 卡片组件

卡片组件是一个多功能的容器，用于展示相关内容和操作。它通常包含标题、描述、内容和底部操作区域。卡片组件广泛用于展示产品、文章、用户信息等内容。

## 导入

```jsx
import { Card, CardHeader, CardFooter, CardTitle, CardDescription, CardContent } from "@/components/ui/card"
```

## 基本用法

<Card className="max-w-md">
  <CardHeader>
    <CardTitle>卡片标题</CardTitle>
    <CardDescription>卡片描述信息</CardDescription>
  </CardHeader>
  <CardContent>
    <p>卡片内容区域，可以包含任何内容，如文本、图像、表单等。</p>
  </CardContent>
  <CardFooter>
    <Button>操作按钮</Button>
  </CardFooter>
</Card>

## 组件 API

### Card

卡片的主容器组件。

| 属性 | 类型 | 默认值 | 描述 |
| --- | --- | --- | --- |
| `className` | `string` | - | 自定义类名 |

### CardHeader

卡片的头部区域，通常包含标题和描述。

| 属性 | 类型 | 默认值 | 描述 |
| --- | --- | --- | --- |
| `className` | `string` | - | 自定义类名 |

### CardTitle

卡片的标题组件。

| 属性 | 类型 | 默认值 | 描述 |
| --- | --- | --- | --- |
| `className` | `string` | - | 自定义类名 |

### CardDescription

卡片的描述组件。

| 属性 | 类型 | 默认值 | 描述 |
| --- | --- | --- | --- |
| `className` | `string` | - | 自定义类名 |

### CardContent

卡片的内容区域。

| 属性 | 类型 | 默认值 | 描述 |
| --- | --- | --- | --- |
| `className` | `string` | - | 自定义类名 |

### CardFooter

卡片的底部区域，通常包含操作按钮。

| 属性 | 类型 | 默认值 | 描述 |
| --- | --- | --- | --- |
| `className` | `string` | - | 自定义类名 |

## 最佳实践

- 使用卡片组件来组织相关的内容和操作
- 保持卡片内容简洁，避免过多的信息
- 使用 `CardHeader`、`CardContent` 和 `CardFooter` 组件来结构化卡片内容
- 在网格布局中使用卡片时，确保卡片高度一致，或使用 `h-full` 类使卡片填充整个网格单元
- 使用 `border-t` 类在卡片的不同部分之间添加分隔线
- 使用 `max-w-*` 类控制卡片的最大宽度，避免卡片在大屏幕上过宽
