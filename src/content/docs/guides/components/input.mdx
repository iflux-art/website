# 输入框组件

输入框组件是一个基础的表单控件，用于收集用户的文本输入。它支持所有标准的 HTML 输入框属性，并提供了一致的样式和行为。

## 导入

```jsx
import { Input } from "@/components/ui/input"
```

## 组件 API

输入框组件接受所有标准的 HTML 输入框属性：

| 属性 | 类型 | 默认值 | 描述 |
| --- | --- | --- | --- |
| `type` | `string` | `"text"` | 输入框类型，如 `text`、`email`、`password` 等 |
| `placeholder` | `string` | - | 输入框的占位文本 |
| `disabled` | `boolean` | `false` | 是否禁用输入框 |
| `required` | `boolean` | `false` | 是否为必填项 |
| `className` | `string` | - | 自定义类名 |

此外，输入框组件还支持所有其他标准的 HTML 输入框属性，如 `value`、`onChange`、`onFocus`、`onBlur` 等。

## 最佳实践

- 始终为输入框提供关联的标签，以提高可访问性
- 使用 `placeholder` 属性提供输入提示，但不要依赖它来替代标签
- 为必填字段添加 `required` 属性，并在视觉上指示它们是必填的
- 使用适当的 `type` 属性，如 `email`、`password`、`number` 等，以获得正确的键盘和验证行为
- 在表单提交前验证输入，并提供清晰的错误消息
- 考虑使用 `aria-invalid` 和 `aria-describedby` 属性来提高可访问性
- 对于复杂的表单，考虑使用表单库，如 React Hook Form 或 Formik
