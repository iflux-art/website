# Textarea 文本域

Textarea 组件用于多行文本输入，如评论、消息或其他需要输入大量文本的场景。

## 基本用法

以下是 Textarea 组件的基本用法示例：

```tsx
import { Textarea } from "@/components/ui/textarea";

export function TextareaDemo() {
  return (
    <Textarea placeholder="在此输入您的消息..." />
  );
}
```

## 禁用状态

可以通过设置 `disabled` 属性来禁用文本域：

```tsx
<Textarea disabled placeholder="禁用状态" />
```

## 带标签

通常与 Label 组件一起使用：

```tsx
import { Label } from "@/components/ui/label";
import { Textarea } from "@/components/ui/textarea";

export function TextareaWithLabel() {
  return (
    <div className="grid w-full gap-1.5">
      <Label htmlFor="message">消息</Label>
      <Textarea id="message" placeholder="在此输入您的消息..." />
    </div>
  );
}
```

## 自定义大小

可以通过 `rows` 属性设置文本域的行数：

```tsx
<Textarea rows={10} placeholder="大文本域..." />
```

## 带描述

添加描述文本以提供更多上下文：

```tsx
import { Label } from "@/components/ui/label";
import { Textarea } from "@/components/ui/textarea";

export function TextareaWithDescription() {
  return (
    <div className="grid w-full gap-1.5">
      <Label htmlFor="message-2">消息</Label>
      <Textarea id="message-2" placeholder="在此输入您的消息..." />
      <p className="text-sm text-muted-foreground">
        您的消息将被发送到我们的团队。
      </p>
    </div>
  );
}
```

## 带验证

结合表单验证使用：

```tsx
import { Label } from "@/components/ui/label";
import { Textarea } from "@/components/ui/textarea";

export function TextareaWithValidation() {
  return (
    <div className="grid w-full gap-1.5">
      <Label htmlFor="message-3" className="text-red-500">
        错误消息
      </Label>
      <Textarea
        id="message-3"
        placeholder="在此输入您的消息..."
        className="border-red-500"
      />
      <p className="text-sm text-red-500">
        请输入有效的消息。
      </p>
    </div>
  );
}
```

## API 参考

Textarea 组件接受所有标准的 HTML textarea 属性。

| 属性 | 类型 | 默认值 | 描述 |
| --- | --- | --- | --- |
| `className` | `string` | - | 自定义类名 |
| `placeholder` | `string` | - | 占位符文本 |
| `disabled` | `boolean` | `false` | 是否禁用文本域 |
| `rows` | `number` | - | 文本域的行数 |
| `value` | `string` | - | 文本域的值（受控模式） |
| `defaultValue` | `string` | - | 文本域的默认值（非受控模式） |
| `onChange` | `(e: React.ChangeEvent<HTMLTextAreaElement>) => void` | - | 值变化时的回调函数 |
