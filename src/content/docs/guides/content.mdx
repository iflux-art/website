### Front Matter

页面头部用 YAML 格式配置基本参数，参数与格式如下：

```mdx
title: 文档/博客标题
excerpt: 文档/博客简介
date: 2025-05-20
tags:
  - 建站
  - 教程
```

### MDX组件规范

1. **组件导入**
   ```jsx
   import { Chart } from '@/components/features/chart'
   import { CodeBlock } from '@/components/ui/code'
   ```
   - 统一通过绝对路径导入
   - 功能组件从`@/components/features/`引入
   - UI组件从`@/components/ui/`引入

2. **交互功能**
   ```jsx
   <CodeBlock 
     language="javascript"
     showLineNumbers
     code={exampleCode}
   />
   ```
   - 需要交互的组件必须使用客户端组件
   - 通过`'use client'`指令声明

3. **代码块使用**
   ````mdx
   ```chart width=600 height=400
   {
     "type": "line",
     "data": {/*...*/}
   }
   ```
   ````
   - 使用自定义Chart组件解析图表语法
   - 通过remark插件实现语法转换
