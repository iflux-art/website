## 什么是事件

在讲故事或写剧本时，要把戏剧性讲（写）出来，戏剧性包括动作、事件和冲突这三个基本元素。

因为某个原因而触发，有某个特定的发展过程，并导致某个结果的一系列事实的完整过程，就是事件。

戏剧的事件，是用简略的方式，将观众带入人物所处的生活场景当中，发生逻辑关联的发展过程。

## 事件的标准

戏剧事件要满足以下标准：

1. 要有足够的冲击力，让人物一下就动起来，有触发、激励、打破宁静、打破常态的作用。
2. 能够引起观众的关注和期待。
3. 能引起一系列的连锁反应，像多米诺骨牌。
4. 能造成人物之间的外部冲突和内心纠结、困惑。
5. 能激发观众的想像空间。

在设计事件时，要用这五个要点来考验一下。没有这五个要件，就不是戏剧事件。

## 事件和事实的区别

事实也叫非事件，生活中会发生很多事情，但事情并不都是事件。

事实是事件的片段，事件是若干个事实有机地、有因果地串连起来的过程。

我们在剧本中要写的是事件，而不是片段的事实，不是没有关联的碎片的事情。

## 事件的种类

1. 封闭事件，相对独立，有头有尾。
2. 开放事件，结尾开放，可以连接下一个事件，成为下一个事件的原因。
3. 被腰斩的事件，发展着发展着突然断了，以后可能在什么地方拣起来再说。
4. 结局上扬的事件。是正面的事件。
5. 结局下沉的事件。如悲剧的事件。

## 设计事件的原则

1. 引爆的原则：要用短暂的、明确的、有冲击力的东西把注意力挑起来，把期望挑起来，在故事中叫激励事件。
2. 变化的原则：一个事件一定要变化，头是一个样子，尾是另一个样子，要有螺旋的上升，波浪的发展，有个大的弧线。
3. 误导与意外的原则：编剧要会设局，误导观众，把观众往沟里带，结果不是结果，完全出乎意料。
4. 放与收的原则：要研究观众的心情、期望值，有时候你要放，放是给予满足他，收就是往回要，夺取，不满足他。不要一味地放，要适当地收。
5. 节奏：事件的发展，要有节奏的变化，快与慢，强与弱，轻与重。
6. 无事生非原则：在一个事件中，借助某个人物的口或手把事端挑起来，没有事儿，这个事就不好看，不成立，不能完成它的使命。
7. 极致性原则：就是通常说的张力，要极致化，要把很多事向极致的方向推，也叫下狠手，不要太仁慈。比如一个人物，你让观众喜欢他，刚刚喜欢这个人，然后你就折磨这个人，给他设计坑、陷阱、危机，打压他、整他，让他痛不欲生、生不如死，让他处于绝境，不要轻易放手、仁慈。只有这样，才能把事件向极致化发展。
8. 天堂之路与地狱之路的原则：有的事件是不断走上坡路，越来越光明，一级比一级高，向天堂走去。但有的事件是倒过来，向地狱走，越走越低、窄，暗。
9. 压弹簧原则：这是个很好的设计事件的技巧。你朝下压，就要弹起来。有两个重要的构成因素，一是反向地压，你希望朝哪个方向发展，就反向地压。二是你给的力量越大，弹起来的力量也越大。具体说，假如你要让人物成功，这是结局，你就让他失败，失败得越深，成功的反弹越高；要让人物悲，前面就让他喜，让他骄傲得不得了，众人都捧他，悲的时候，就很有力量。最核心的技巧，就是倒着使劲。

## 事件的组合形式

要把一个大的故事分解成若干个事件来讲，把相互关联的情节组合起来，构成一个文艺作品。

1. 单元式事件组合：《西游记》师徒三人走到一个地方，突然一阵风，妖怪出来了，最终事件解决；过一段又重复这个事，一件一件，最终到了西天。这就是单元式的。
2. 首尾叠进式组合：第一件事没结束，第二件事又开始了，产生关联，第一个事件的果很可能成为第二个事件的因。
3. 平行或交替式组合：两件事同时发展，交替着出现。
4. 分叉式组合：一个案件发生了，交给两个行动组去做，两个组的任务不一样，事件发展的方向也不一样。
5. 归一式组合：若干个事件，发生之后往前发展，突然在某个点上合二为一或合三为一。
6. 终点转起点的组合：首尾相叠，只是形式上的重叠，但有的时候你可以从内在联结它，这个事件的发生与发展形成结尾，这个事件迅速引发第二个事的起点，因果转换。
7. 俄罗斯套娃式组合：大事件套小事件，多个事件层层套在一起。
8. 魔幻之门式组合：故事很神秘，走不出去了，突然找到一个门，打开门之后，进入一个完全不一样的世界，从一个事件跳到另一个事件。
9. 游戏通关式组合：进了一个关，下面还有一关，一关比一关难打。
