随着视频号的不断发展，越来越多的创作者加入其中，为平台带来了丰富多样的内容。为了维护良好的平台生态，保障广大用户的观看体验，特制定本视频号内容规范。请各位创作者在发布作品前，仔细阅读并遵守这些规范。

## 发布作品前，检查有没有这些问题

### 营销导流

营销导流行为会降低观众的观看体验，此类行为包括但不限于以下几种情况：
| 问题 | 说明 | 详情 |
| ---- | ---- | ---- |
| 营销导流 | 降低观众的观看体验 | 1. 视频中带联系方式、二维码<br />2. 视频标题外链标题<br />3. 账号名带联系方式<br />4. 无法保证安全的交易风险<br />5. 视频不完整，引导跳转到未经验证的链接 |

平台鼓励创作者通过优质内容吸引观众，而不是通过不当的营销导流手段。一旦发现有营销导流行为，平台将根据情节严重程度进行相应处理。

### 无资质发文

在认证领域发布专业的内容时，一定要确保自身具备相应资质，不要辜负观众的信任。具体要求如下：
| 问题 | 说明 | 详情 |
| ---- | ---- | ---- |
| 无资质发文 | 在认证领域发布专业的内容，<br />不要辜负观众的信任 | 1. 未认证健康类专业资质时，不能发布涉及医疗、保健等生命健康领域的内容；无论是否认证，不能发布医疗保健相关的营销推广内容<br />2. 发布财经内容前，要获得对应的资质认证，未经专业验证的内容，可能对观众的健康、财产带来损失 |

创作者应尊重观众的知情权和选择权，确保发布的专业内容真实、可靠。

### 违背公序良俗

违背公序良俗的内容不仅会影响平台的形象，也会对社会风气产生不良影响。公序良俗是社会基本的道德规范和行为准则，创作者应自觉遵守。
| 问题 | 说明 | 详情 |
| ---- | ---- | ---- |
| 违背公序良俗 | 违背家庭伦理道德 |  |

任何违背公序良俗的作品都将无法在平台发布，严重者将被追究相应责任。

### 骗粉

真诚是与观众建立良好关系的基础，真诚对待你的观众，更有可能获得他们的持续关注。骗粉行为会大大降低粉丝对你的信任。
| 问题 | 说明 | 详情 |
| ---- | ---- | ---- |
| 骗粉 | 真诚对待你的观众，更有可能获得他们的持续关注 | 视频中诱导观众关注、点赞、评论等欺骗粉丝的行为，会大大降低粉丝对你的信任 |

平台倡导创作者通过优质内容和良好的互动来吸引和留住粉丝，而不是通过骗粉手段。

## 注意避免影响视频观看体验的问题

### 画面和声音

观众观看视频时，画面和声音的质量直接影响观看体验。请检查画面是否有卡顿、黑屏、颠倒180°、声音和画面不同步、台词字幕和拍摄画面无关等问题。
| 问题 | 详情 |
| ---- | ---- |
| 画面和声音 | 检查画面是否有卡顿、黑屏、颠倒180°、声音和画面不同步、台词字幕和拍摄画面无关 |

创作者应在发布前对视频进行充分的检查和优化，确保观众能够获得良好的观看体验。

### 模板视频

模板视频往往缺乏创意和个性，容易让观众产生审美疲劳。为了给观众带来新鲜、独特的内容，建议创作者避免套用动画编辑软件模板，创作画面粗糙的视频（简单画面+鸡汤文字）。用心创作你独特的内容，更容易受到观众的喜爱。
| 问题 | 详情 |
| ---- | ---- |
| 模板视频 | 避免套用动画编辑软件模板，画面粗糙的视频（简单画面+鸡汤文字），用心创作你独特的内容，更容易受到观众的喜爱 |

平台鼓励创作者发挥自己的创意和才华，打造出具有个人特色的优质作品。

### 旧闻

发布与发布日期间隔已久的日期、明显的过期热点等旧闻内容，可能会让观众觉得内容缺乏时效性和价值。请尽量发布新鲜、有价值的内容，满足观众的信息需求。
| 问题 | 详情 |
| ---- | ---- |
| 旧闻 | 与发布日期间隔已久的日期、明显的过期热点 |

创作者应关注时事热点，及时创作并发布具有时效性的内容，提升作品的吸引力。

## 结语

遵守视频号内容规范是每个创作者的责任和义务。希望各位创作者能够共同维护平台的良好生态，创作出更多优质、有价值的作品。平台将持续对违规行为进行监督和处理，为广大用户提供一个健康、积极的视频观看环境。
