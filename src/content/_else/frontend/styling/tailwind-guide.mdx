# Tailwind CSS 完整指南：从基础到高级应用

Tailwind CSS 是一个实用优先的 CSS 框架，它提供了大量的实用类来快速构建现代化的用户界面。本指南将带你从基础概念到高级应用全面掌握 Tailwind CSS。

## 第一部分：Tailwind CSS 基础

### 1.1 什么是 Tailwind CSS

Tailwind CSS 是一个实用优先（utility-first）的 CSS 框架，它提供了大量的原子级 CSS 类，让你可以直接在 HTML 中构建设计。

**核心理念**：
- **实用优先**：使用小而专一的实用类
- **响应式设计**：内置响应式修饰符
- **可定制性**：高度可配置的设计系统
- **性能优化**：只包含使用的样式

### 1.2 安装和配置

**通过 npm 安装**：
```bash
npm install -D tailwindcss
npx tailwindcss init
```

**基础配置文件**：
```javascript
// tailwind.config.js
/** @type {import('tailwindcss').Config} */
module.exports = {
  content: ["./src/**/*.{html,js,jsx,ts,tsx}"],
  theme: {
    extend: {},
  },
  plugins: [],
}
```

**在 CSS 中引入**：
```css
/* styles.css */
@tailwind base;
@tailwind components;
@tailwind utilities;
```

### 1.3 基础实用类

**间距系统**：
```html
<!-- 内边距 -->
<div class="p-4">padding: 1rem</div>
<div class="px-6 py-3">padding: 0.75rem 1.5rem</div>
<div class="pt-2 pr-4 pb-6 pl-8">各方向不同内边距</div>

<!-- 外边距 -->
<div class="m-4">margin: 1rem</div>
<div class="mx-auto">水平居中</div>
<div class="mt-8 mb-4">上下外边距</div>
```

**颜色系统**：
```html
<!-- 文字颜色 -->
<p class="text-gray-900">深灰色文字</p>
<p class="text-blue-600">蓝色文字</p>
<p class="text-red-500">红色文字</p>

<!-- 背景颜色 -->
<div class="bg-white">白色背景</div>
<div class="bg-gray-100">浅灰背景</div>
<div class="bg-blue-500">蓝色背景</div>

<!-- 边框颜色 -->
<div class="border border-gray-300">灰色边框</div>
```

**字体和文本**：
```html
<!-- 字体大小 -->
<h1 class="text-4xl">大标题</h1>
<h2 class="text-2xl">中标题</h2>
<p class="text-base">正文</p>
<small class="text-sm">小字</small>

<!-- 字体粗细 -->
<p class="font-light">细体</p>
<p class="font-normal">正常</p>
<p class="font-bold">粗体</p>

<!-- 文本对齐 -->
<p class="text-left">左对齐</p>
<p class="text-center">居中</p>
<p class="text-right">右对齐</p>
```

## 第二部分：布局和响应式设计

### 2.1 Flexbox 布局

```html
<!-- 基础 Flex 容器 -->
<div class="flex">
  <div class="flex-1">项目 1</div>
  <div class="flex-1">项目 2</div>
  <div class="flex-1">项目 3</div>
</div>

<!-- Flex 方向和对齐 -->
<div class="flex flex-col items-center justify-center h-screen">
  <h1 class="text-2xl mb-4">垂直居中</h1>
  <button class="px-4 py-2 bg-blue-500 text-white rounded">按钮</button>
</div>

<!-- 响应式 Flex -->
<div class="flex flex-col md:flex-row gap-4">
  <div class="flex-1">内容 1</div>
  <div class="flex-1">内容 2</div>
</div>
```

### 2.2 Grid 布局

```html
<!-- 基础网格 -->
<div class="grid grid-cols-3 gap-4">
  <div class="bg-gray-200 p-4">项目 1</div>
  <div class="bg-gray-200 p-4">项目 2</div>
  <div class="bg-gray-200 p-4">项目 3</div>
</div>

<!-- 响应式网格 -->
<div class="grid grid-cols-1 md:grid-cols-2 lg:grid-cols-3 gap-6">
  <div class="bg-white p-6 rounded-lg shadow">卡片 1</div>
  <div class="bg-white p-6 rounded-lg shadow">卡片 2</div>
  <div class="bg-white p-6 rounded-lg shadow">卡片 3</div>
</div>

<!-- 复杂网格布局 -->
<div class="grid grid-cols-4 grid-rows-3 gap-4 h-96">
  <div class="col-span-2 row-span-2 bg-blue-200">主要内容</div>
  <div class="bg-green-200">侧边栏 1</div>
  <div class="bg-green-200">侧边栏 2</div>
  <div class="col-span-4 bg-gray-200">页脚</div>
</div>
```

### 2.3 响应式设计

**断点系统**：
```html
<!-- 响应式类名 -->
<div class="w-full md:w-1/2 lg:w-1/3 xl:w-1/4">
  响应式宽度
</div>

<!-- 响应式显示/隐藏 -->
<div class="block md:hidden">移动端显示</div>
<div class="hidden md:block">桌面端显示</div>

<!-- 响应式文字大小 -->
<h1 class="text-2xl md:text-4xl lg:text-6xl">
  响应式标题
</h1>
```

**移动优先设计**：
```html
<div class="
  p-4 
  text-center 
  bg-white 
  sm:p-6 
  sm:text-left 
  md:p-8 
  lg:p-12 
  xl:p-16
">
  移动优先的响应式设计
</div>
```

## 第四部分：Typography 插件

### 4.1 安装和配置

```bash
npm install -D @tailwindcss/typography
```

```javascript
// tailwind.config.js
module.exports = {
  plugins: [
    require('@tailwindcss/typography'),
  ],
}
```

### 4.2 基础用法

```html
<article class="prose lg:prose-xl mx-auto">
  <h1>文章标题</h1>
  <p>这是一个使用 Typography 插件的文章段落...</p>
  <h2>子标题</h2>
  <ul>
    <li>列表项 1</li>
    <li>列表项 2</li>
  </ul>
</article>
```

### 4.3 自定义 Typography

```javascript
// tailwind.config.js
module.exports = {
  theme: {
    extend: {
      typography: {
        DEFAULT: {
          css: {
            color: '#333',
            maxWidth: 'none',
            h1: {
              color: '#111',
              fontWeight: '800',
            },
            h2: {
              color: '#111',
              fontWeight: '700',
            },
            a: {
              color: '#3182ce',
              '&:hover': {
                color: '#2c5282',
              },
            },
          },
        },
      },
    },
  },
}
```
