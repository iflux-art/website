容器化部署（Containerization）通过将应用程序及其依赖打包为标准化的容器镜像，实现环境一致性和快速部署。

主流的容器化部署工具包括 Docker、Kubernetes 等。

## Docker 命令

### 镜像管理

```bash
docker images
```

```bash
docker rmi image_id
```

```bash
docker system prune -a
```

### 容器操作

```bash
docker ps -a
```

```bash
docker start/stop container_id
```
