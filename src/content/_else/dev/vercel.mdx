## 解决 Vercel 在国内无法访问的问题

### 修改 A 记录

在域名解析时，将 Vercel 默认的 `A` 记录 `76.76.21.21` 更改为 `76.223.126.88` 或 `76.76.21.98` 。

### 修改 CNAME 记录

在域名解析时，将 Vercel 默认的 `CNAME` 记录 `cname.vercel-dns.com` 更改为 `cname-china.vercel-dns.com` 。
