# 发布作品前，检查有没有这些问题

| 问题         | 说明                                              | 详情                                                                                                                                                                                                                    |
| ------------ | ------------------------------------------------- | ----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| 营销导流     | 降低观众的观看体验                                | 1. 视频中带联系方式、二维码 <br>2. 视频标题外链标题 <br>3. 账号名带联系方式 <br>4. 无法保证安全的交易风险 <br>5. 视频不完整，引导跳转到未经验证的链接                                                                   |
| 无资质发文   | 在认证领域发布专业的内容， <br>不要辜负观众的信任 | 1. 未认证健康类专业资质时，不能发布涉及医疗、保健等生命健康领域的内容；无论是否认证，不能发布医疗保健相关的营销推广内容 <br>2. 发布财经内容前，要获得对应的资质认证，未经专业验证的内容，可能对观众的健康、财产带来损失 |
| 违背公序良俗 | 违背家庭伦理道德                                  |                                                                                                                                                                                                                         |
| 骗粉         | 真诚对待你的观众，更有可能获得他们的持续关注      | 视频中诱导观众关注、点赞、评论等欺骗粉丝的行为，会大大降低粉丝对你的信任                                                                                                                                                |

# 注意避免影响视频观看体验的问题

| 问题       | 详情                                                                                                      |
| ---------- | --------------------------------------------------------------------------------------------------------- |
| 画面和声音 | 检查画面是否有卡顿、黑屏、颠倒180°、声音和画面不同步、台词字幕和拍摄画面无关                              |
| 模板视频   | 避免套用动画编辑软件模板，画面粗糙的视频（简单画面+鸡汤文字），用心创作你独特的内容，更容易受到观众的喜爱 |
| 旧闻       | 与发布日期间隔已久的日期、明显的过期热点                                                                  |
