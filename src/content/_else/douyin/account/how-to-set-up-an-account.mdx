账号搭建分为静态搭建和动态搭建。

## 静态搭建

包含昵称、简介、头像、头图、视频封面等。

### 昵称

#### 要求

- 品牌名称露出、行业关键词体现、内容价值凸显。
- 好理解、好记忆、好传播。

#### 作用

- 突出人设
- 突出账号功能

### 简介

#### 目的

- 传递信息/价值，增加粉丝信任度，并且让粉丝知道关注你能获得什么。
- 尽量有温度，避免无关闲言碎语。

#### 角度

- 我是谁
- 我的产品特点是什么
- 关注我你能得到什么
- 我的直播时间
- 企业号链接

> 普通人如果没有高大上的履历，可以写个人经历。

### 头像

#### 类型

- 人物IP：形象照、生活照、艺术照等，画质清晰、有亲和力。
- 品牌IP：品牌形象、品牌LOGO等。

#### 要求

- 图片清晰：抖音头像的标准尺寸是500\*500。
- 有辨识度：如果你有职业，最好通过头像，第一眼就知道你是做什么的，比如说医生。
- 头像昵称一致化，IP形象具象化。
- 真实：有人设尽量真人出镜，用自己的照片，不要用与自己形象不符的图片。
- 有温度、有亲和力。
- 品牌商家类，直接用企业logo或者企业名称就行。

### 头图

#### 要求

是账号主页最显眼的地方，根据自身情况选择适配类型，整体风格与账号契合。

#### 类型

| **类型**   | **具体内容**                       | **适合人群**      |
| ---------- | ---------------------------------- | ----------------- |
| 主题背景   | 根据主题设计背景图片               | 蓝V/品牌/机构     |
| 照片背景   | 单人照/合照                        | 旅行/VLOG情侣账号 |
| 求关注背景 | 引导关注                           | 种草/泛娱乐/萌宠  |
| 小贴士背景 | 如何购买产品、成为会员等引导性内容 | 服饰/美妆/配饰等  |
| 简介背景   | 个人介绍/微博封面                  | 人物IP/体现价值   |

### 视频封面

- 人设号：突出出境人设，加深观众记忆度，封面带上视频关键词，更能激发观看兴趣。
- 连续剧集：封面保持系列感，整体观感上统一。

## 动态搭建

包括拍摄手法、剪辑方式、人物表现、商品展示、直播间搭建等。
