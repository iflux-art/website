## 2025年4月6日

技术选型，确定技术栈和网站基本架构：

- 基础库：React + react-dom（最新稳定版）
- 前端框架：Next.js + next-themes（App Router），用于构建高性能的 React 应用
- CSS 框架：Tailwind CSS v4D，实用优先的 CSS 框架
- 组件库：Radix UI + ShadcnUI 组件库，提供可访问的 UI 组件
- 图标：lucide-react，统一的图标风格
- 包管理器：pnpm
- Node.js 版本：20.11.1 或更高
- 部署工具：GitHub + Vercel，用于部署静态网站和服务器端渲染应用
